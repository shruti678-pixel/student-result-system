# 🎓 Student Result System

A simple web-based application that calculates student grades based on marks.

## 🚀 Features
- Input student name and marks
- Automatically calculates grade
- Displays pass/fail status
- Clean and modern UI
- Input validation

## 📊 Grading System
- 90+ → Grade A
- 75–89 → Grade B
- 50–74 → Grade C
- Below 50 → Fail

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🌐 Live Demo
https://result-star-app.lovable.app

## 📌 How to Run
1. Download or clone the repository
2. Open `index.html` in browser

## 💡 Project Description
This project demonstrates basic conditional logic and front-end development skills by building a simple and interactive Student Result System.
